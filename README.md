# Ultimate Reverse Shell #

Reverse shell implementation using Python.
<br/><br/>


## Features ##
### Implemented ###
* Auto reconnection
* Downloader (Download file to target machine)
* Screenshot
* Full control on the target machine.

### Not Implemented ###
* Packet encoder/decoder. (Not planning on it!)
